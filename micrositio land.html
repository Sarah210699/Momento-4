Tipografía

Para el logo se emplearon dos tipos de tipografía, para el nombre de la academia se utilizó Unica One y para el segundo texto Rajdhani. Asimismo para los títulos y el menú de navegación se usó Abel, para el cuerpo de texto se usó Montserrat. Todas las fuentes fueron extraídas de Google Fonts.

Diseñadores

Unica One: Eduardo Tunni
Rajdhani: Indian Type Foundry
Abel: MadType
Montserrat: Julieta Ulanovsky
